# Pacific Trails Resort Website

A responsive static website for Pacific Trails Resort, showcasing luxury lodging experiences on the California North Coast.

## Description

Pacific Trails Resort offers a unique lodging experience with private yurts overlooking the Pacific Ocean. This website provides information about accommodations, activities, and reservations for visitors interested in nature and luxury. This is a project that I've created while I was taking a class at the local community college

## Features

- **Home Page**: Introduction to the resort with video tour and amenities overview
- **Yurts**: Detailed information about yurt accommodations
- **Activities**: Available activities and guided tours
- **Reservations**: Contact information and booking details
- **Responsive Design**: Optimized for both desktop and mobile viewing
- **Accessibility**: Includes HTML5 semantic elements and proper meta tags

## Technologies Used

- HTML5
- CSS3
- Responsive web design techniques
- Video and image media integration

## How to View

Since this is a static website, you can view it by:

1. Opening any of the HTML files (e.g., `index.html`) directly in a web browser
2. Or serving the files through a local web server

For a local server, you can use Python:
```bash
python -m http.server 8000
```
Then navigate to `http://localhost:8000` in your browser.

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Includes conditional comments for Internet Explorer 9+ support
- Responsive design works on mobile devices
